Internal annotation guideline

Dictionary word annotation process

  1. If a word found in ‘বাংলা-বানান অভিধান’ (hard copy), then it should be taken as correct (Printing errors are exceptional).
  2. If the spelling of the word is different from the one in the ‘বাংলা-বানান অভিধান’, then the word in the ‘বাংলা-বানান অভিধান’ is correct, the spelling in the rest of the dictionary should be treated as wrong.
  3. If a word is not found in the ‘বাংলা-বানান অভিধান’, but found in the ‘আধুনিক বাংলা অভিধান’; In that case the ‘আধুনিক বাংলা অভিধান’ spelling will be considered correct.
  4. If there is a word that is not in ‘বাংলা-বানান অভিধান’ or not found in ‘আধুনিক বাংলা অভিধান’, then the latest spelling rules of Bangla Academy should be followed for getting correct spelling.
  5. If the correct spelling of a word cannot be resolved by the above rules, one should refrain from deciding on that word.
  6. If dictionary indicates as error (অশুদ্ধ কিন্তু প্রচলিত) and excluded spelling (বর্জিত বানান) should treat as incorrect but if indicates as only conventional spelling (প্রচলিত বানান) should treat as correct.

Non-dictionary word annotation process

  1. Should follow Bangla Academy’s latest rules for verifiyng a word and for providing a correction.
  2. If Bangla Academy suggests more option to write particular words, then annotator should keep both as right form.
  3. If any word found that has no instruction by Bangla Academy, then should follow Bangla Academy’s latest dictionary, books or universally accepted institution or Language related books' rules.
  4. For new words, should make decisions depends on Languages structure, custom, grammatical formation etc. Then justify by semantic-based method. That particular word is frequently used or not— it also be justified.
  5. No symbols and emojis are treated as a part of a word token.
  6. If found any word with affix, should check first base word correct or not and then should verify the use of affix and finally should check the combination is correct or not.
  7. Most frequently used recognized language related books, well known newspapers, writings and websites' new word should accept as correct word if does not violate latest Bangla Academy's rules.
  8. Single character should detect as invalid.

Named Entity annotation process

  1. For places, institutions & others should follow government & institutional spelling.
  2. For foreign name entity should keep internationally recognized (English form) & native language spelling both.
  3. For prominent figure should follow his/her name’s recognized spelling.

Foreign Word annotation process

  1. For Foreign Word Should follow Bangla Academy’s latest rule & dictionaries.
  2. Most commonly used words which have no entry in dictionary should keep spelling following pronounce instructed by Bangla Academy.
  3. If in any words used internationally recognized (English form) & native language spelling, should keep both.

Correction providing process

Correction of a non-word should be provided following the latest Bangla Academy’s spelling rules and following dictionary hierarchy–

  1. If a word found in ‘বাংলা-বানান অভিধান’ (hard copy), then it should be taken as correct (Printing errors are exceptional).
  2. If the spelling of the word is different from the one in the ‘বাংলা-বানান অভিধান’, then the word in the ‘বাংলা-বানান অভিধান’ is correct, the spelling in the rest of the dictionary should be treated as wrong.
  3. If a word is not found in the ‘বাংলা-বানান অভিধান’, but found in the ‘আধুনিক বাংলা অভিধান’; In that case the ‘আধুনিক বাংলা অভিধান’ spelling will be considered correct.
  4. If there is a word that is not in ‘বাংলা-বানান অভিধান’ or not found in ‘আধুনিক বাংলা অভিধান’, then the latest spelling rules of Bangla Academy should be followed for getting correct spelling.
  5. If the correct spelling of a word cannot be resolved by the above rules, one should refrain from deciding on that word.
  6. Correction of non-word should provide expecting lowest edit distance (Lowest to highest transformation- like, অপরা < অপর/অপেরা/অপরাধ/অপরাস্ত/অপরাহ্ন) and should ignore relatively highest edit distance based corrections .

Use Case annotation process

  1. If a word token used as a name (Name of a place, person and organization), should be marked as a Named Entity, if used as general word, should be marked as a general vocabulary and if used as both Named Entity & general word, should be marked as both case (Named Entity and general vocabulary).

Suffix annotation process

  1. In case a word used as only suffix then should provide correction starting with hyphen (like :-তে).
  2. If suffix used as both general vocabulary/named entity and suffix should provide both types of corrections, starting with hyphen and without hyphen.

Invalid annotation process

  1. If any word token does not mean anything or a fragmented portion of a word, it should be marked as invalid.
  2. If a non-word or a fragmented portion of a word is unable to correct applying the lowest edit distance, it should be marked as a invalid word.

Special cases annotation process

  1. Whether a word token is existent or non-existent in Banan Ovidhan, Adhunik Ovidhan, or any other dictionaries: if the word token follows the latest Bangla Academy’s spelling rules, it should be marked as a correct word, otherwise a non-word error.
  2. Bangla Academy suggests, one can write last অ-ধ্বনি as ও-কার following pronounce in any word
  3. So, everyone has choice to write last letter ও-কার by following pronounce or not
  4. We keep both form as correct
  5. Only forward slash (/) and hyphen (-), can be used between two valid word tokens.
  6. Only single dot (.) and apostrophe (') can be used as a character of a valid word token.
  7. Normally visarga (ঃ) is not allowed at the end of Bangla word but if found in Banan or Adhunik ovidhan can be treated as correct.
  8. As visarga (ঃ) is not allowed at the end of Bangla word that can be a colon (:) based on text context.

Sentence annotation process for Grammar solution

  1. Bangla Academy 'Standard Bangla Practical Grammar' (প্রমিত বাংলা ব্যাবহারিক ব্যাকরণ) and Bangla Academy’s others latest rules should follow in identifying and correcting grammatical mistakes in sentences.
  2. If any cases there has no direct instruction from Bangla Academy annotator should follow recognized grammar books and keep its reference.
  3. Should check from various angle if the sentence is correct or incorrect.
  4. Should give priority if any linguistic or grammatical way to keep sentences as correct.
  5. Only should mark as incorrect & give reason/error categories, when it is really incorrect by language structure and custom.
  6. Should identify multiple grammar error and provide correction is if necessary.
  7. Some errors can be interpreted as multiple grammatical errors, in this case, the most acceptable error should mark as an error and should provide correction is accordingly.
  8. Emojis, symbols, and foreign words should avoid.
  9. Run-on errors (Non-space errors), Split word errors (One-space errors) and real word errors should treat as Grammar errors.
